# Orme PWA

App per la gestione dei luoghi scout del gruppo Turi 1.

## Comandi

### Installazione
```bash
npm install
```

### Avvio Sviluppo
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Struttura
- `src/components`: Componenti UI riutilizzabili
- `src/pages`: Pagine principali (Home, Profile, etc.)
- `src/lib`: Logica di business (data, gamification, proposals)
- `src/types`: Definizioni TypeScript
